Servidor REST

Servidor REST con propósitos educativos y de enseñanza.

Escrito por @@EruJazz, financiado por @@jjcardonao.

Paso 1

Inicializa la base de datos con /db/create , esto creará las tablas necesarias en una pequeña base de datos en Sqlite (puedes eliminar las tablas con /db/drop).

Paso 2

el API es como sigue:

  • PUT /user

    Crea un nuevo usuario. Recibe un json como el siguiente (Nota: al crear el usuario no importa el número de id. El username debe ser único):

    {
    id:0,
    username: "pepz",
    nombre: "Pepipto Perez",
    android_id: "codigoLaaaaaaaargoQueIdentificaUnDispositivoAndroid9876543210"
    }

    El sistema generará automaticamente al usuario y retornará el user_id.

  • GET /user/:id

    Retorna los datos de un usuario, buscandolo por su id

  • GET /user/:un

    Retorna los datos de un usuario, buscandolo por su username

  • DELETE /user/:id

    Elimina un usuario, haciendolo por su id.

  • POST /user/gcm/:id

    Actualiza el código GCM del despositivo de un usuario, id es el id del usuario. Además se debe enviar un json como el que sigue.

    {
    android_id: "codigoLaaaaaaaargoQueIdentificaUnDispositivoAndroid9876543210"
    }

  • PUT /pet

    Crea una nueva mascota, el json que se envía debe ser como el siguiente (donde dueno es la clave foranea al dueño de la mascota, y 'longitud, latitud' son las coordenadas proyectadas en WGS84-4326):

    {
    id:0,
    dueno: 1,
    nombre:"Rex",
    edad: 2,
    longitud: 6.27018,
    latitud: -75.598095
    }

    Retorna el pet_id de la mascota creada.

  • POST /pet/:id

    Actualiza la posición de una mascota. El POST debe enviar un json así:

    {
    lon: 6.27018,
    lat: -75.598095
    }

    Al hacer la actualización, envía una notificación push al dueño de la mascota.

  • GET /pet/:id

    Retorna los datos de una mascota, buscandola por su id

  • DELETE /pet/:id

    Elimina una mascota, haciendolo por su id

  • GET /gcm/:gcm_id

    Envía una notificación push de prueba al dispositivo con el gcm_id que se le ponga.
